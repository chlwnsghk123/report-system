# 📝 학습 리포트 생성기

서버 설치 없이 `index.html` 더블클릭만으로 실행되는 학습 리포트 도구입니다.

---

## 🚀 사용 방법

1. `index.html`을 크롬(Chrome) 브라우저로 열기
2. **엑셀 파일 불러오기** 클릭 → `학습리포트_샘플데이터.xlsx` 선택
3. 상단에서 **수업일** 선택
4. 탭에서 **학생** 선택
5. 저번 주차 과제 이행 상태 입력
6. 필요 시 **미니 테스트 / 코멘트** 체크박스 활성화
7. **PDF로 저장** 버튼 클릭

---

## 📁 파일 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 앱 본체 (서버 불필요, 크롬에서 열기) |
| `학습리포트_샘플데이터.xlsx` | 샘플 데이터 (학생 6명 × 16주) |
| `README.md` | 이 파일 |

---

## 📊 엑셀 파일 구조

### 시트 1 — 수업정보
| 열 | 내용 |
|----|------|
| 날짜 | YYYY-MM-DD 형식 (예: 2026-03-03) |
| 강사명 | 강사 이름 |
| 전체문제수 | 미니테스트 전체 문제 수 |
| 교재 | 교재명 |
| 단원 | 단원명 |
| 상세진도 | 상세 진도 내용 |
| 과제1~5 | 이번 주 과제 항목 |

### 시트 2 — 성적
- 학생별 날짜별 점수, 맞힌 개수, 오답 번호 관리

### 시트 3 — 과제현황
- 날짜별 학생별 이행률(%) 및 과제1~5 상태 (완료 / 부분완료 / 미완료)
- 각 날짜 구분 행에 과제 내용이 미리 표시됨

---

## ✅ 주요 기능

- **학생 탭** — 학생별로 탭 전환하며 독립적으로 입력
- **숙제 이행률 그래프** — 최근 6회 이행률 꺾은선 그래프
- **미니 테스트** (선택) — 맞힌 수 / 전체 문제 + 오답 태그
- **코멘트** (선택) — 선생님 한 줄 코멘트
- **PDF 저장** — 가로 A4에 2페이지 나란히 (시험자료 첨부 가능)
- **자동 저장** — 브라우저 IndexedDB에 자동 저장, 재방문 시 복원

---

## 💡 팁

- 날짜는 **화요일 / 토요일** 패턴 (변경 가능)
- 수업일 추가 시 엑셀 `수업정보` 시트에 행 추가 후 다시 불러오기
- 학생 추가 시 `성적` 시트에 이름 행 추가
- PDF 저장 전 **엑셀 저장**을 먼저 권장
