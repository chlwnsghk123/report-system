# 요청 방법 가이드

> 파일명을 몰라도 됩니다. 하고 싶은 걸 그냥 말하면 됩니다.
> AI가 CLAUDE.md의 라우팅 표를 보고 스스로 파일을 찾습니다.

---

## 이렇게 말하면 됩니다

### 위치·순서 바꾸기
```
시험자료 첨부 버튼을 미니 테스트 항목 안으로 넣어줘.
```
```
저장 버튼을 PDF 저장 버튼 아래로 옮겨줘.
```
```
이행률 입력란을 과제 에디터 위로 올려줘.
```

### 색상·디자인 바꾸기
```
버튼들 색을 전부 파란색(#2563eb)으로 바꿔줘.
```
```
리포트카드 주색상을 파란색으로 바꿔줘.
```
```
학생 탭 폰트 크기 좀 키워줘.
```

### 기능 동작 바꾸기
```
그래프에 최근 6주치가 보이게 해줘. 지금은 4개밖에 안 보여.
```
```
점수 계산을 맞힌 수 / 전체 * 100으로 바꿔줘.
```
```
엑셀 불러올 때 강사명 자동입력 기능 없애줘.
```
```
미니 테스트가 기본으로 켜져 있게 해줘.
```

### 새 기능 추가하기
```
코멘트 섹션에 '다음 목표' 입력란 하나 더 추가해줘.
```
```
리포트카드에 수업 날짜 대신 '몇 번째 수업인지' 도 같이 보여줘.
```

### 버그 수정
```
엑셀 저장하면 이행률이 0으로 초기화되는 버그 고쳐줘.
```
```
탭 전환하면 오답 번호가 사라지는 문제 고쳐줘.
```

## 팁

**더 정확하게 말하고 싶을 때** — 어느 화면인지 구분해주면 좋습니다.
```
왼쪽 패널에서 미니 테스트 토글 위치를 코멘트 토글 아래로 바꿔줘.
```
```
리포트카드(오른쪽)에서 저번 주차 과제 섹션 배경색 바꿔줘.
```

**여러 개를 한 번에** — 한 번에 한 가지씩 요청하면 실수가 줄어듭니다.
```
(좋음) 첨부 버튼을 미니 테스트 안으로 넣어줘.
(주의) 첨부 버튼 위치 바꾸고 색도 바꾸고 PDF 버튼도 없애줘. → 실수 위험
```

**동작이 이상할 때** — 어떤 상황에서 어떻게 이상한지 말해주세요.
```
엑셀 불러오고 날짜 선택하면 진도가 자동입력이 안 돼.
```
```
PDF 저장 누르면 리포트 카드가 빈 페이지로 나와.
```
